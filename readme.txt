Adds a "robots" meta tag to the head of your site
 
== Description ==

This is a very simple plugin that adds a "robots" meta tag to the head of your site.  You can toggle "noindex", "nofollow", and "noarchive".

== Installation ==

1. Upload the simple-robots-meta folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
